# ホルムズ海峡、開いてる？

ホルムズ海峡が開いているかどうかだけを確認するアプリ。**全自動。人間は触らない。**

## 動く仕組み

```
GitHub Actions (30分ごと)
  └ collect.py
      ├ UKMTO / MARAD / NGA の航行警報を取得
      ├ 米・イランの公式発表からホルムズ言及を抽出
      ├ 見出しを機械翻訳（DeepL → 落ちたら Claude Haiku）
      ├ ブレント原油価格を取得
      ├ ルールエンジンで警戒レベルを判定
      └ docs/status.json を書き出してコミット
            ↓
      GitHub Pages が docs/ を配信
            ↓
      index.html が status.json を読んで描画（5分ごとに再取得）
```

サーバーは要りません。無料枠だけで回ります。

## セットアップ

1. このリポジトリを push
2. **Settings → Pages** で `main` ブランチの `/docs` を公開
3. **Settings → Secrets → Actions** に翻訳キーを登録（任意）
   - `DEEPL_KEY` … DeepL API（無料枠あり。推奨）
   - `ANTHROPIC_API_KEY` … DeepLが落ちたときの予備
   - どちらも無ければ、翻訳せず原文のまま表示します
4. **Actions** タブで `update` を一度手動実行
5. `collect.py` の `rules_url` を自分のリポジトリURLに書き換える

## このアプリが守っていること

**1. 「情報が無い」を「平常」の根拠にしない**  
情報源が2つ以上生きていることを確認できたときだけ判定します。取れなければ「営業中」ではなく **「確認中」**。

**2. 根拠の種類を隠さない**  
通航実績（AIS）は計測していません。判定の根拠は「どの当局も閉鎖を報告していない」ことだけです。画面にそう書いてあります。

これが成立するのはホルムズ海峡だからです。世界の石油の約2割が通る海峡の閉鎖が、UKMTO・米海事局・海運メディアの全てに報じられないことはあり得ない。**閉鎖は静かには起きない。**

**3. 発言を創作しない**  
吹き出しは公式発表の見出しそのもの。機械翻訳したうえで、**原文・出典リンク・「自動翻訳」バッジ**を必ず併記します。

**4. 判定にLLMを使わない**  
警戒レベルは `decide()` のルールだけで決まります。再現性と説明可能性が命なので。生成AIは翻訳にしか使いません。

**5. 重要度は「通航に影響したか」で決める**  
声の大きさでは決めません。基準は `severity()` に書いてあり、アプリからリンクしています。

**6. 有事にはふざけるのをやめる**  
警戒レベルが上がると、政治家の顔は灰色になり、口を閉じ、海から船が消えます。
